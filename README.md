# BYU Linguistics Mentoring Archive

A two-part archive for the BYU Department of Linguistics:

- **Faculty Mentoring** — mentor-submitted records of research projects,
  conference presentations, and student-development outcomes.
- **Student Experiences** — student reflections on transformative
  mentored experiences, organized by the four BYU Aims.

## Quick Start

1. See `DEPLOYMENT_GUIDE.txt` for what to upload where.
2. See `CHANGES.md` for what was modified in this update.
3. The site is designed for GitHub Pages — just push these files

This site follows the official BYU brand guidelines published at

ling.byu.edu. Specifically:

- IBM Plex Sans typography
- Navy / white / royal palette (no other colors)
- Official BYU Linguistics sub-brand logos (no custom typeset)
- Header / nav / footer structure matching the BYU web theme

## Privacy Notes

Each entry can be marked private at the field level or at the
entry level. Private entries do not appear in the public archive
but are visible in the admin view. Anonymous students appear as
"Anonymous student" publicly; their real names are visible to admins.

The admin password is a client-side soft barrier — fine for
casual privacy, NOT real security. For sensitive personal data
you would need NetID/SSO.
