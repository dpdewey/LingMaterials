============================================================
  WHY EACH FOLDER IS SELF-CONTAINED
============================================================

Each subfolder includes its own copy of styles.css, storage.js,
and seed-data.js. This means:
  - You can open any URL directly without depending on root files
  - GitHub Pages serves each page correctly with no path issues
  - The cost is a few duplicated files (totalling ~80KB)

The duplication is intentional. Don't be alarmed by it.
